# Ejecutar con Docker

## Descargar imagen desde Docker Hub

```bash
docker pull jikuo/landing-jahir
```

---

## Ejecutar contenedor

```bash
docker run -d -p 8080:80 jikuo/landing-jahir
```

---

## Abrir en navegador

```plaintext
http://localhost:8080
```

---

# Construcción Manual de la Imagen

## Construir imagen Docker

```bash
docker build -t landing-jahir .
```

---

## Ejecutar contenedor manualmente

```bash
docker run -d -p 8080:80 landing-jahir
```
